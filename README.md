# Модель детектора BBC в Geant4  
  
#### Атвор: Завидов Егор  

#### Для скачивания:
```
git clone https://github.com/victorsemipalatin/BBC.git
```
Модель построена с использованием пакета CADMesh, который позволяет создавать тела при инициализации геометрии
напрямую из stl-файлов, созданных в САПР. Для полученного тела доступны любые
стандартные процедуры в Geant4.
